Benvindo ao QuickApp Desenv - Cooperchip

Este modelo de aplicativo fornece uma maneira rápida e conveniente de criar um ASP.NET Core 5.0 / Angular 11 applications

NOTA: Para contratar desenvolvedores para o seu projeto, vá para DevLink ou envie um Email para devlink@ebenmonney.com para maiores informações.
Tecnologias
  • Template com Angular 11 e TypeScript
  • RESTful API Backend usando ASP.NET Core 5.0 MVC Web API
  • Banco de Dados usando Entity Framework Core
  • Autenticação baseda em IdentityServer4 e ASP.NET Core Identity
  • Documentação da API usando Swagger
  • Contrução do Client-side no Angular CLI
  • Tema usando Bootstrap 4 e Bootswatch
Features
  • Uma estrutura de projeto de back-end e front-end completa, com login, usuário e gerenciamento de funções baseada em permissão integrada
  • Código Angular / ASP.NET Core Clean e simples para servir de guia
  • Camada de Acesso a Dados construída com o Padrão Repositório e Unit of Work
  • Code First Database
  • Uma Design RESTful API
  • Serviços de Dialog e Notification
  • Página de Configuração e Serviço de Configuração
  • Internacionalização Integrada
  • Thema com SASS
  • Preparado para usar API de email
  • Manipulador de Access e Refresh Tokens com WebStorage (Bearer authentication) - sem Cookies
  • CRUD na APIs
  • etc.
